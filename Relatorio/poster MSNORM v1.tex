  \LHead{Introdução}
  
Modularidade é uma propriedade desejável de sistemas de reescrita porque permite que um sistema combinado herde as propriedades dos seus componentes. Terminação não é modular, mesmo assim sobre certas restrições modularidade pode ser recuperada. Nesse trabalho, é apresentado uma formalização do Teorema de Normalização Forte Modular no assistente de provas Coq. A prova segue as ideias da tese de PhD do Lengrand \cite{Lengrand}, mas acredita-se que essa é a primeira formalização deste teorema.

As contribuições deste trabalho podem ser resumidas em:
\begin{itemize}
\item Foi construída uma prova construtiva do Teorema de Normalização Forte Modular, e
\item foi provado a equivalência entre a definição do Lengrand de normalização forte e a definição indutiva padrão de normalização forte. 
\end{itemize} 	
 
\vspace{0.12cm}
 \LHead{O Teorema de Normalização Forte Modular}

  Uma relação de um conjunto $A$ para si mesmo é uma relação de redução sobre um conjunto, i.e. uma relação de redução sobre $A$ é um subconjunto de $A \times A$. Uma relação de um conjunto para si mesmo é um {\itshape relação de redução sobre um conjunto}, i.e. uma relação de redução sobre A é um subconjunto $A \times A$. Se $\rightarrow_A$ é uma relação de redução sobre A, então uma {\itshape sequência de redução} é uma sequência da forma $a_0 \rightarrow_A a_1 \rightarrow_A a_2
    \rightarrow_A \ldots$. Uma sequência de redução $a_0 \rightarrow_A a_1 \rightarrow_A a_2
    \rightarrow_A \ldots \rightarrow_A a_n \, (n\geq 0)$ é umm $n$-passos redução de $a_0$. Uma sequência de redução é finita se ela é uma redução em $n$-passos para algum $n \in \mathbb{N}$, e infinita caso contrário. Nós escrevemos $\rightarrow_A^+$ (resp. $\rightarrow_A^*$) para o fecho transitivo (resp. reflexivo e transitivo) de $\rightarrow_A$.
    Um elemento $a \in A$ é fortemente normalizado w.r.t. $\rightarrow_A$ se toda sequência de redução começando de $a$ é finita, e neste caso nós escrevemos $a \in {SN}^{\rightarrow_A}$. Normalmente, essa ideia é expressada indutivamente como se segue: $$a \in {SN}^{\rightarrow_A} \mbox{ sse } \forall b, (a \rightarrow_A b \mbox{ implica b } \in {SN}^{\rightarrow_A})$$ 
Para apresentar o teorema, nós precisamos definir as noções de simulação forte e fraca. Nas seguintes definições $A$ e $B$ são conjuntos arbitrários.

    Seja $\rightarrow$ uma relação de 
     $A$ para $B$, $\rightarrow_A$ uma relação de redução sobre $A$ e $\rightarrow_B$
     uma relação de redução sobre $B$. A relação de redução $\rightarrow_B$
     {\it fortemente} (resp. {\it fracamente}) simula $\rightarrow_A$ através de
     $\rightarrow$ se $(\leftarrow \# \rightarrow_A) \subseteq (\rightarrow_B^+ \#
     \leftarrow)$ (resp. $(\leftarrow \# \rightarrow_A) \subseteq (\rightarrow_B^*
     \# \leftarrow)$).

 Seja $\rightarrow$ uma relação de $A$ para $B$, $\rightarrow_1$ e
      $\rightarrow_2$ duas relações de redução em $A$, e $\rightarrow_B$ uma relação de redução em $B$. Suponha que:
      \begin{enumerate}
        \item $\rightarrow_B$ simula fortemente $\rightarrow_1$
                             através de $\rightarrow$;
        \item $\rightarrow_B$ simula fracamente $\rightarrow_2$ através de $\rightarrow$;
        \item $A \subseteq SN^{\rightarrow_2}$.
      \end{enumerate}

      Então $\leftarrow ({SN}^{\rightarrow_B}) \subseteq {SN}^{\rightarrow_1\cup\rightarrow_2}$. Em outras palavras, $$\forall a:A, a\in \leftarrow({SN}^{\rightarrow_B}) \mbox{ implica } a \in {SN}^{\rightarrow_1 \cup \rightarrow_2}.$$

\LHead{Resultados}
\normalfont

Lengrand usa a seguinte definição para normalização forte: \newline
{\color{blue}
$Definition SN \{A : Type\} \{red : Red A\} (a : A) : Prop := \newline
\forall P,\, patriarchal\, red\, P \rightarrow P\, a$.
}

Neste trabalho foi usada a definição indutiva e foi provado a equivalência entre as duas definições: \newline
{\color{blue}
Definition SN' \{A : Type\} \{red : Red A\} (a : A) : Prop := \newline
$sn\_acc: (\forall b,\, red\,a\, b\, \rightarrow SN'\, red b) \rightarrow SN'\, red\, a$.
}

O teorema seguinte {\itshape SNbySimul} é conhecido como normalização forte por simulação. O teorema afirma que se uma relação de redução sobre A, digamos {\itshape redA}, é fortemente simulada por uma relação de redução sobre B, digamos {\itshape redB}, através de R então a pré-imagem de qualquer elemento que satisfaz o predicado $(SN' redB)$ também satisfaz $(SN' redA)$. \newline
{\color{blue}
Theorem SNbySimul \{A B\} \{redA : Red A\} \{redB: Red B\} \{R: Rel A B\} : \newline
$StrongSimul\, redA\, redB\, R \rightarrow \forall a,\, Imagem (inverse R) (SN' redB)\, a \newline \rightarrow 
SN' redA\, a$.
}

O lema {\itshape SNunion}, que é um resultado também utilizado na prova do teorema principal, dá uma caracterização do predicado $SN' (redA !\_! red'A)$. 
Uma propriedade importante usada é a chamada estabilidade. Nós dizemos que um predicado P é estável w.r.t. a relação de redução R quando, para todo $a$ e $b$ tal que $R\,a\, b$, $P\, a \mbox { implica } P\, b$. 
%Sob a hipótese de estabilidade de $(SN'\, redA)$ w.r.t. a relação de redução $red'A$, o predicado $SN'(redA !\_! red'A)$  pode então ser decomposto na conjunção $(SN' ((refltrans redA) \# red'A)) \land (SN' redA)$.

O Teorema de Normalização Forte Modular é specificado na sintaxe de Coq como abaixo: \newline
{\color{blue}
Theorem ModStrNorm \{A B: Type\} \{redA red'A: Red A\} \{redB: Red B\} \{R: Rel A B\} : \newline
$(StrongSimul\, red'A\, redB\, R) \rightarrow (WeakSimul\, redA\, redB\, R) \rightarrow \newline (\forall b: A,\, SN'\, redA\, b) \rightarrow \forall a: A,\, Imagem (inverse R) (SN' redB)\, a \rightarrow \newline SN' (redA !\_! red'A)\, a$.
}
\normalfont

 \LHead{Conclusão}
 
  A prova formalizada é construtiva, no sentido que não depende da lógica clássica, o que é interessante do ponto de vista computacional devido ao conteúdo algorítmico correspondente das provas. Provas construtivas são normalmente mais difíceis e elaboradas do que as clássicas, mas são mais preferíveis no contexto da ciência da computação. 
 
O teorema da normalização forte modular é um resultado abstrato que diz as condições para a união de duas relações de redução que preservam a normalização forte. Esse teorema é, por exemplo, aplicado em \cite{Substi} para estabelecer a propriedade PSN de um cálculo com substituições explícitas.
